# Changelog

All notable changes to SubstackSaver will be documented in this file.

## [1.0.0] - 2026-03-13

### Added
- One-click save button with quick-tag dropdown
- Reading list dashboard (opens in new tab)
- Progress tracking (auto-saves reading position)
- Color-coded tags and folders
- Search and filter by author, tags, title
- Edge Collections export/import
- Dark/light mode support
- Right-click context menu integration
- Edge sync for cross-device access

### Technical
- Manifest V3 extension
- Fluent Design System styling
- Security: XSS protection via input sanitization
- Storage: sync with local fallback

### Added
